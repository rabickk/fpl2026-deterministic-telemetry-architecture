# FPL 2026 Deterministic Telemetry Core

This repository contains the VHDL source code and supporting materials for the paper:

"A Hardware-Centric Deterministic Telemetry Architecture for FPGA-Based Real-Time Systems"

The artifact includes:
- VHDL source files
- XDC constraints
- Simulation and implementation results
- Power and utilization reports
- Documentation

Target platform:
- Digilent Nexys A7-100T
- Xilinx Artix-7 XC7A100T
- AMD Vivado 2023.2
